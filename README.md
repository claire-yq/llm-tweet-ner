Project comparing high-demand vs. low-demand NER on small language models, using data from Tweebank-NER.

## Setup
Each `.ipynb` file contains all code used to fine-tune and evaluate the model. Results were obtained using a T4 High-RAM GPU on Google Colab. Installation requirements are at the beginning of the script.

`f1.py` contains extra code to calculate regular and per-class F1.
